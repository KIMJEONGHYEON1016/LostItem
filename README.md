## 전체적인 소개

첫 프로젝트로 만들게 된 어플입니다. <br>
LostItem 프로젝트를 시작하게 된 계기는 이미 대중화돼서 사용되고 있지 않으며,<br>
공부하기위해 여러가지 기능들을 포함하고 있는 어플을 만들고 싶었습니다.<br><br>
이 어플은 물건을 잃어버리거나 분실물을 습득한 사용자가 보다 원활하게 물건을 찾을 수 있게 도와주는 어플입니다. <br>
**IOS**를 기반으로 **swift**를 사용하여 제작하였습니다.<br>
핵심적으로 동작할 지도는 **NaverMap**을 사용하였으며, 데이터를 저장하는 것은 **Firebase**를 사용했습니다.<br>

## 화면 소개 및 기능 

### 로그인
<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/585abe0e-864a-409f-a601-fbd734ebd004" width="400" ><br>

- 애플 로그인, 카카오 로그인, 구글 로그인을 지원합니다. 해당 소셜에서 이메일 정보를 가져옵니다.

- 한 번 로그인 할 시 로그인 정보를 저장해 재 로그인할 필요는 없습니다. 

### 메인화면

<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/c06c90cf-5fab-4990-913e-2f72cb0eeb13" width="400" ><br>
- **NaverMap**을 사용했고, 기본적인 기능으로는 **현재 위치로 이동**, **지도 축소 및 확대 기능**과 별도로 추가한 **분실물 마커 기능**,<br>
**지도 새로고침 기능**을 추가했습니다.<br>

- 아래 탭바를 이용해서 다른 메뉴로 이동이 가능합니다.<br><br>

### 게시글
<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/b1ed41ab-bee7-4129-b70a-d42693a71a30" width="400" ><br>

- 사진을 첨부하여 게시글을 작성할 수 있습니다.<br>

- 제목, 사진(최소 1장), 내용 이 세가지 중 한가지라도 작성이 안 되어있을시 게시글을 등록할 수 없습니다.<br>

- 작성한 게시글을 메인화면에서 분실물 마커로 표시해 해당 위치에 다른 사용자가 확인할 수 있게 합니다.<br>

<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/99ad0675-347a-4173-b8af-fe8c06e43a31" width="400" height="800" ><br>
- 완료 버튼을 누르면 화면 중앙에 표시된 마커 위치에 마커가 생성됩니다.<br><br>

### 채팅
<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/e69519dd-efd3-4285-9acd-98bffbb74155" width="400" ><br>
- 메인화면에 나타난 마커를 사용자가 터치하게 될 시, 분실물의 상세정보를 확인 가능합니다.

- 채팅 버튼을 통해서 게시글 작성자에게 채팅이 가능합니다.<br><br>

<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/12b153e7-f06a-48f0-b270-ebcef7d291d9" width="400" height="800" ><br>

- 마커에서 채팅버튼을 이용해 채팅하는 화면입니다.

<img src="https://github.com/KIMJEONGHYEON1016/LostItem/assets/127102729/db85b6f9-2023-46b8-b188-e027eb80e8a4" width="400" ><br>
- 메인화면 탭바를 통해 채팅으로 이동시, 채팅방 목록이 표시됩니다. 터치하며 진입하여 채팅이 가능합니다.
